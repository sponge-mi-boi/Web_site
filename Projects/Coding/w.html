Alpha-Generation Backtesting Framework

This is a documentation and description of my Python-based signal generation and analysis framework to create alpha-generating (α Gen) strategies based on time-series data of various stocks in the SP 500.

This provides an overview of the approach, including list of methods that will be experimented, the tools that will be used, and the approaches which will be looked at.
