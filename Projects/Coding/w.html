Alpha-Generation Backtesting Framework

This is a documentation and description of my Python-based signal generation and analysis framework to create alpha-generating (α Gen) strategies based on time-series data of various stocks in the SP 500. The initial focus is on OOP design for efficiency, but different classes of strategies and methods of portfolio allocation will be tested and analyzed later.
