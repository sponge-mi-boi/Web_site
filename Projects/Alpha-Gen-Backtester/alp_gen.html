Alpha-Generation Backtesting Framework

This is a documentation and description of the creation and implementation of my Python-based signal generation and analysis framework to create alpha-generating strategies based on time-series data of various stocks in the SP 500.

This provides an overview of the approach such as the tools that will be used, and the approaches which will be analyzed.

Function based Framework: Part 1

Version 1.0

The operations are fully vectorized with the use of Dataframes instead of series with lists. The result is a working model of backtesting which was done cointegrated pairs as filtered through the ADF test, in the SP 500. Parallel processing is also used to improve performance and parquet files are used for storage. Walk forward testing is used based upon a series of 5 tests and there is also classification of various stocks. Also, graphs are given of the highest performing pairs in the example time period. Parameter optimization will be done next, along with validation testing.

Cointegration is done on daily data, with the goal being to conduct a rolling forward, validating backtest which is replicable. Future goals are to do this over several periods to find patterns related to cointegrated assets and to find optimal parameters to a greater degree of accuracy.
